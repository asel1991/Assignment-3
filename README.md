# Assignment-3

## О проекте

Этот проект реализует систему управления личным банковским счетом на Java. Вы можете:

- Вносить деньги.
- Снимать деньги.
- Просматривать историю транзакций.
- Проверять текущий баланс.

## Классы в проекте

1. **TransactionType**: Перечисление для типов транзакций (DEPOSIT и WITHDRAWAL).
2. **Amount**: Класс для хранения суммы транзакции и её типа.
3. **PersonalAccount**: Класс для управления банковским счетом.
